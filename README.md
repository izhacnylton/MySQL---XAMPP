## ProtoTech_DB

Sistema de controle de estoque e vendas de componentes eletrônicos.

## Objetivo 
Gerenciar produtos, vendas e estoque, permitindo análise de dados e automação de processos.

## Funcionalidades

- Cadastro de produtos e categorias  
- Controle de estoque  
- Registro de vendas  
- Relatórios de faturamento
